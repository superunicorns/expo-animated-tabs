## Expo Animated Tabs
a nice animated tabs built with expo.

![app screenshot](./app_screenshot.png)

### Built with

- Expo
- React Native
- Reanimated
- Moti

### Demo
[video demo](https://www.youtube.com/watch?v=p7PEK7uewzI)

### Available Scripts

First of all clone this repository.

In the project directory, you can run:

- step 1: run `npm install` (wait untill all node-modules are in place) 

- step 2: run `npm start` (open the project in an emulator or a real mobile)
